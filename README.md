# End-to-End Telecom Customer Churn Prediction
## Project Overview
This project focuses on predicting customer churn in the telecom industry using Machine Learning techniques. The goal is to identify customers who are likely to leave the service and help businesses take proactive retention measures.
In addition to churn prediction, customer segmentation is performed using K-Means Clustering to identify different customer groups based on their behavior and service usage patterns.
## Problem Statement
Customer churn is one of the major challenges faced by telecom companies. Acquiring new customers is significantly more expensive than retaining existing ones. This project aims to build a predictive model that can identify potential churners and provide actionable business insights.
## Dataset
The dataset contains customer demographic information, account details, service subscriptions, and churn status.
